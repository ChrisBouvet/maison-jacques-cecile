# Photos — Appartement 2e étage

Déposez ici les photos avec ces noms exacts pour qu'elles apparaissent automatiquement sur la page :

- `principale.jpg`  — vue panoramique (grande, en haut de la galerie)
- `sejour.jpg`      — séjour lumineux
- `cuisine.jpg`     — cuisine ouverte
- `chambre-1.jpg`   — chambre 1 (vue montagne)
- `chambre-2.jpg`   — chambre 2

💡 Pour la vue sur le village (ou toute autre pièce), utilisez les emplacements "Plus de photos" (`extra-1.jpg`, etc. — voir plus bas).

Format recommandé : JPEG, au moins 1200 px de large pour la photo principale, 800 px pour les autres.

Tant qu'un fichier n'est pas présent, l'encadré "Photo à venir" correspondant reste affiché.

## Ajouter plus de photos

Pour ajouter d'autres photos au-delà des 5 emplacements ci-dessus, déposez-les dans ce dossier sous le nom :

- `extra-1.jpg`
- `extra-2.jpg`
- `extra-3.jpg`
- ... jusqu'à `extra-30.jpg`

Vous pouvez en ajouter autant que vous voulez (jusqu'à 30), dans n'importe quel ordre, et avec des "trous" dans la numérotation si besoin (par exemple seulement extra-1 et extra-5 — les deux s'afficheront, triées par numéro).

Elles apparaîtront automatiquement dans une section "Plus de photos" sous la galerie principale, avec un agrandissement au clic.
